# Mallo 프로젝트 구조 규칙

## 폴더 구조

- **화면(페이지) 컴포넌트**: `src/screens/` 폴더
  - 예: `HomeScreen.jsx`, `RecordScreen.jsx`, `ProfileScreen.jsx`

- **재사용 UI 컴포넌트**: `src/components/`

- **커스텀 훅**: `src/hooks/`

- **스타일 관련 공통 컴포넌트**: `src/components/ui/` (있으면)

## 네이밍 규칙

- 새로 만드는 화면(Screen) 컴포넌트는 `src/screens/` 안에 넣고 파일 이름은 `SomethingScreen.jsx` 형태로 만들어야 합니다.
